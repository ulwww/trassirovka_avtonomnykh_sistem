Использование: python main.py адрес
Например:
    python main.py www.google.com
    python main.py 108.177.14.105

Краткое описание: отправляю ICMP сообщения с соответствующим TTL,
получаю ответы и их обрабатываю, потом обращаюсь к сайту https://ipinfo.io,
от кторого получаю информацию об автономной системе, стране и провайдере.
